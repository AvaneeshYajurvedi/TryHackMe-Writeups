#  TryHackMe Writeups

Welcome to my TryHackMe journey — a growing collection of hands-on cybersecurity labs, writeups, and lessons learned along the way.

##  About This Repository

This repository contains detailed writeups for rooms I’ve completed on TryHackMe.
Each writeup focuses on:

*  Enumeration techniques
*  Exploitation methods
*  Privilege escalation
*  Key concepts and takeaways

The goal is not just to solve rooms, but to understand the *why* behind every step.
